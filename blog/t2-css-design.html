Absolute, Relative, Fixed Positioning

By Wei Wen August 11, 2015.

Positioning is one of the most basic yet important concepts in CSS language. New learners often get confused by different positioning propeties: static, relative, absolute, and fixed. It is perfectly normal. What we need to know is positioning is not as hard as it seems to be. Next, I will explain what each positioning property means.

Static. This is default position for every element. When you create a div(section), the default postion is the static position. Normally, no one will use static positioning because the position cannot be manipulated. Instead, people will set positioning to relative, absolute, or fixed.

Relative. This type positioning positions the element relative to itself. It is the most confusing and misused positioning for new learners. However, it is very straightforward once you understand how it works. When you set an element relative positioning without setting top, bottom, left, and right, it will stay on the default position. Once you change the attributes such as top: 10px, the element will shift down by 10px from its original position.

Absolute. This type positioning is one of the most powerful tools that allows you position any element anywhere accurately. It positions the elements according to the positions of its next parent elements with relative(absolute) positioning. If there is no such parent, the element will be placed relatively to the whole page itself. What we need to pay attention is that once we set absoulte positioning, it will not be affected by other elements and will not affect other elements. Therefore, it could be hard to control as more and more elements added to the page if we overuse absolute positioning.

Fixed. This type positioning hold an elecment static relatively to viewport or the browser window itself. If you use fixed positioning to an element placed on the left corner of viewport, its position will be consistent regardless of scroliing up or down. Fixed positioning was used more often in the past, but it is rarely used in modern website design.

I hope this article can help new learners better understand positioning concept and the importance of it.
